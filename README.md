✨ Основные функции  
🎯 Высокоточная временная шкала: Сетка времени на основе CSS Grid, позволяющая перетаскивать и выравнивать блоки действий с точностью до 1 кадра.
🔄 Интерактивное перетаскивание: Плавное перетаскивание навыков с помощью Vue.Draggable, поддержка перемещения из библиотеки навыков на дорожку и их корректировки внутри дорожки.
🔗 Визуализация связей: Динамическое рисование кривых Безье через SVG для отображения связей и зависимостей между навыками в реальном времени.
👥 Управление несколькими персонажами: Поддержка динамического переключения оперативников на дорожках, строгая логика размещения для обеспечения точности действий.
🛠️ Встроенный редактор данных: Визуальный JSON-редактор для удобного обслуживания атрибутов оперативников и данных навыков.

🛠️ Технологический стек  
Проект построен на современном фронтенд-стеке:  

Фреймворк: Vue 3 (Composition API)  

Сборщик: Vite  

Управление состоянием: Pinia  

Библиотека UI-компонентов: Element Plus  

Библиотека для перетаскивания: Vue.Draggable  

Стили: CSS Grid + CSS Variables  

🔧 Инструменты для разработчиков  
В проект встроен скрытый редактор данных для создания файла gamedata.json.  

Адрес доступа:  

Локальная разработка: http://localhost:5173/editor  

Инструкция по использованию:  

Перейдите по указанному адресу.  

На этой странице можно добавлять новых оперативников, изменять атрибуты навыков (CD, потребление, множители и т.д.).  

Нажмите кнопку «Сгенерировать и скачать», затем замените файл public/gamedata.json в проекте скачанным файлом для обновления данных.  

🚀 Локальная разработка  
Если вы хотите запустить проект локально или участвовать в разработке:  

Требования к окружению:  

Node.js (рекомендуется v16 или выше)  

npm или yarn  

Установка зависимостей:  

bash  
npm install  
Запуск сервера разработки:  

bash  
npm run dev  
После запуска перейдите по адресу http://localhost:5173, чтобы увидеть интерфейс планировщика.
